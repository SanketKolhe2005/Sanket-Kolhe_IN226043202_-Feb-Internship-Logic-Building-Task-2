# Logic Building Task – 2  
Data Science Internship (February 2026)

This repository contains my solutions for Logic Building Task – 2 assigned during the Data Science Internship – February 2026.

## About This Task

The objective of this task is to improve basic Python programming and logical thinking skills.  
The problems focus on working with:

- Lists
- Sets
- Dictionaries
- Tuples
- Loops and conditional statements

All solutions are implemented in Python using Jupyter Notebook.

## Problems Included

The notebook covers the following problems:

1. Finding unique words in a sentence  
2. Identifying the highest salary from employee data  
3. Finding maximum and minimum values in a list  
4. Counting products above a certain price  
5. Calculating attendance percentage  
6. Removing duplicate phone numbers  
7. Counting character frequency in a string  
8. Converting a list into a tuple  
9. Checking if a key exists in a dictionary  
10. Calculating average marks  

## Purpose of This Work

These problems help in understanding how basic programming concepts are applied in real-world scenarios such as:

- Text processing
- HR and payroll systems
- Data analysis
- Attendance management
- Customer data handling
- Student performance evaluation

- Python 3
